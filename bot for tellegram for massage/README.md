# Telegram-бот для запису на масаж

Бот допомагає клієнтам дізнатися ціни, записатися на масаж і отримати відповіді на часті запитання. Масажист отримує заявку в Telegram із ім'ям, послугою, бажаним часом і телефоном.

Усі тексти боту — українською.

## Запуск за 5 хвилин

### 1. Встановіть Python 3.11 або 3.12

Python 3.14 поки не підходить: деякі залежності вимагають Visual Studio Build Tools.

### 2. Створіть і активуйте віртуальне середовище

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3. Встановіть залежності

```powershell
pip install -r requirements.txt
```

### 4. Заповніть конфігурацію

Скопіюйте `.env.example` у `.env`:

```powershell
Copy-Item .env.example .env
```

Відкрийте `.env` і заповніть:

```env
BOT_TOKEN=токен_від_BotFather
ADMIN_CHAT_ID=ваш_telegram_id
TIMEZONE=Europe/Kyiv
DATABASE_PATH=massage_bot.sqlite3
BUSINESS_ADDRESS=адреса кабінету
MAP_URL=https://maps.google.com/...
PENDING_REQUEST_TTL_MINUTES=120
```

### 5. Запустіть бота

```powershell
python -m massage_bot.main
```

## Як дізнатися свій ADMIN_CHAT_ID

Напишіть боту `@userinfobot` у Telegram — він покаже ваш Telegram ID. Запишіть це число в `.env` як `ADMIN_CHAT_ID`.

## Можливості MVP

- Клієнтське меню: Послуги та ціни / Записатися / Адреса і графік / Інше питання
- Розділ **Мої записи** зі статусом і можливістю скасування
- Запис: послуга → додаткові послуги (за бажанням) → день → вільний час → ім'я → телефон → підтвердження
- **Додаткові послуги** (апсейл): мультивибір; можуть додавати ціну та час (або лише ціну). Час допослуг враховується у розрахунку вільних слотів
- Валідація номера телефону (формат +380XXXXXXXXX або 0XXXXXXXXX)
- Нова заявка одразу з'являється в адмін-панелі (з допослугами, підсумком і контактом клієнта в один тап)
- Адміністратор може підтвердити, відхилити, перенести або скасувати запис
- Клієнт може скасувати підтверджений запис
- FAQ по 4 темах: оплата, тривалість, протипоказання, перенесення
- Автоматичне нагадування через 60 хвилин, якщо клієнт не завершив запис
- Автоматичне нагадування за 24 години до підтвердженого сеансу
- Непідтверджена заявка автоматично звільняє слот через 120 хвилин
- Нерозпізнані повідомлення пересилаються масажисту
- SQLite база без зовнішніх сервісів

## Адмін-панель (для масажиста)

Доступна лише для адміністраторів із `.env`. Кнопки з'являються після першого `/start`.

Редагування — кнопкове, без введення тексту з `;`:

- **Послуги** — оберіть послугу й змінюйте кнопками: назву (одним повідомленням), тривалість і ціну (готові варіанти або «Інша сума»). «Додати послугу» — 3 кроки: назва → тривалість → ціна.
- **Додаткові послуги** — той самий редактор. Тривалість допослуги може бути `0` (тоді вона додає лише ціну, не впливає на час сеансу).
- **Графік роботи** — оберіть день тижня й налаштуйте кнопками: початок, кінець (із сітки часу) та «Зробити вихідним/робочим». Окремо можна закрити чи відкрити конкретну дату (відпустка, свято).

У розділі **Адреса і контакти** можна змінити адресу кабінету та посилання на карту без перезапуску бота.
